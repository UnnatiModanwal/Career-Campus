## 🚀 How to Start Your Application

### Quick Start (Recommended)
```bash
Double-click: start_all.bat
```
This will start both backend and frontend automatically!

### Manual Start
```bash
# Terminal 1: Backend
cd backend
.\venv\Scripts\Activate.ps1
python start_server.py

# Terminal 2: Frontend  
cd frontend
npm start
```
