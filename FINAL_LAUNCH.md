## 🔧 **TROUBLESHOOTING**

### If something doesn't work:

1. **Backend Issues**:
   ```bash
   cd backend
   .\venv\Scripts\Activate.ps1
   python simple_server.py
   ```

2. **Frontend Issues**:
   ```bash
   cd frontend  
   npm start
   ```

3. **Check URLs**:
   - Frontend: http://localhost:3000
   - Backend: http://localhost:8000/health
